# C++ Practice

A collection of C++ programs covering Data Structures, Algorithms, and Competitive Programming problems - from fundamentals to advanced topics.

## Topics Covered

| Category | Topics |
|---|---|
| Data Structures | Arrays, Linked Lists, Stacks, Queues, Trees, Graphs, Heaps, Hash Maps |
| Algorithms | Sorting, Searching, Recursion, Backtracking, Divide & Conquer |
| Dynamic Programming | Memoization, Tabulation, Classic DP problems |
| STL | Vectors, Maps, Sets, Priority Queues, Iterators |
| Competitive Programming | Codeforces / LeetCode style problems |

## How to Run

Make sure you have a C++ compiler installed (g++ recommended).

```bash
# Compile
g++ -o output filename.cpp

# Run
./output
```
